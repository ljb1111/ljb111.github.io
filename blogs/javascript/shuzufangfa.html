    # forEach

    forEach() 方法对数组的每个元素运行一次给定的函数。

    arr.forEach(callback(currentValue [, index [, array]])[, thisArg])
    1

    其中：

    callback 是每个元素执行的函数，它接受以下三个参数：

    • currentValue

      当前正在处理中的元素

    • index 可选

      当前元素在原数组中的索引

    • array 可选

      forEach()方法正在操作的数组。

    # 实例

    const items = ['item1', 'item2', 'item3'];
const copy = [];
items.forEach(function(item){
  copy.push(item);
});

    # filter

    filter 方法创建一个新数组, 新数组包含通过回调函数的所有元素。

    var newArray = arr.filter(callback(element[, index[, array]]){
// Return element for new_array 
}[, thisArg]) //不改变原数组，返回过滤后的新数组。
    1
    2
    3

    其中：

    callback函数用来测试每个元素，如果函数返回值为true，则保留该元素，返回值为false，则不保留该元素，它接受以下三个参数：

    • element

      当前正在处理中的元素（callback函数参数）

    • index 可选

      当前元素在原数组中的索引

    • array 可选

      调用 filter 的数组本身

    # 实例

    function isBigEnough(element) {
  return element >= 10;
}
var filtered = [12, 5, 8, 130, 44].filter(isBigEnough);
// filtered is [12, 130, 44]

    # map

    map方法创建一个新数组，新数组元素是原数组的每个元素调用函数的返回值。

    var new_array = arr.map(function callback(currentValue[, index[, array]]) {
// Return element for new_array 
}[, thisArg])//不改变原数组，返回过滤后的新数组。
    1

    其中：

    callback生成新的数组，它接受以下三个参数：

    • element

      当前正在处理中的元素（callback函数参数）

    • index 可选

      当前元素在原数组中的索引

    • array 可选

      调用 map 的数组本身

    # 实例

var roots = numbers.map(Math.sqrt);

    # reduce

    reduce() 方法对数组中的每个元素执行一个提供的函数(升序执行)，将其结果汇总为单个返回值。

    arr.reduce(callback(accumulator, currentValue[, index[, array]])[, initialValue])
    1

    其中：

    callback执行数组中每个值 (如果没有提供 initialValue则第一个值除外（因为如果没有初始值，那么 reduce 会将数组的第一个元素作为初始值，并从第二个元素开始迭代）)的函数，包含四个参数：

    • accumulator

      累计器累计回调的返回值; 它是上一次调用回调时返回的累积值，或initialValue

    • currentValue

      数组中正在处理的元素

    • index 可选

      数组中正在处理的当前元素的索引。 如果提供了initialValue，则起始索引号为0，否则从索引1起始。

    • array 可选

      调用reduce()的数组

    • **initialValue ** 可选

      作为第一次调用 callback函数时的第一个参数的值。 如果没有提供初始值，则将使用数组中的第一个元素。 在没有初始值的空数组上调用 reduce 将报错。

    # 实例

    var total = [ 0, 1, 2, 3 ].reduce(
  ( acc, cur ) => acc + cur,
  0
);

    allNames[name]++;

    // 数字数组去重
let arr = [1,2,1,2,3,5,4,5,3,4,4,4,4];
let result = arr.sort((a,b)=>a-b).reduce((init, current) => {
    if(init.length === 0 || init[init.length-1] !== current) {
        init.push(current);
    }
    return init;
}, []);
console.log(result); //[1,2,3,4,5]

let myArray = ['a', 'b', 'a', 'b', 'c', 'e', 'e', 'c', 'd', 'd', 'd', 'd']
let myOrderedArray = myArray.reduce(function (accumulator, currentValue) {
  if (accumulator.indexOf(currentValue) === -1) {
    accumulator.push(currentValue)
  }
  return accumulator
}, [])
console.log(myOrderedArray)//['a','b','c','e','d']

    # 方法中 thisArg 参数

    几乎所有调用函数的数组方法 —— 比如 find，filter，map，除了 sort 是一个特例，都接受一个可选的附加参数 thisArg。

    let army = {
  minAge: 18,
  maxAge: 27,
  canJoin(user) {
    return user.age >= this.minAge && user.age < this.maxAge;
  }
};

let users = [
  {age: 16},
  {age: 20},
  {age: 23},
  {age: 30}
];

// 找到 army.canJoin 返回 true 的 user
let soldiers = users.filter(army.canJoin, army);

alert(soldiers.length); // 2
alert(soldiers[0].age); // 20
alert(soldiers[1].age); // 23
